Introduction

Data collection

We have collected data from IMDB which is a database of all the cinematographic records in the entire world. We have combined this database with the TMDB API which allows us to obtain information about the revenue and budget of each movie and some information about Oscars Awards to analyze that is useful to analyze the success of movies. The TMDB API could only provide us information about the revenue and budget of around ten thousand movies. For this research question, we narrowed down to only movies and we excluded tv series, documentaries, tv shows, etc.
